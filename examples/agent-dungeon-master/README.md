# Agent Dungeon Master examples

These examples show the expected shape of play for `agent-dungeon-master`.

They use original setting material and avoid copied rulebook text, stat blocks, tables, boxed text, adventure prose, or protected setting content.

- `solo-campaign-start.md` - a campaign opener for a solo player.
- `live-play-turn.md` - good and bad examples of an interactive DM turn.
