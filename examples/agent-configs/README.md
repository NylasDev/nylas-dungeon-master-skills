# Agent config examples

These files show how to point specific agents at the Dungeon Master skills without copying protected content or creating agent-specific forks of the same skill.

- `openclaw.example.jsonc` shows an OpenClaw `openclaw.json` fragment that loads this repository's skill folder as an extra directory and allowlists both skills.

Do not commit real user config, API keys, or machine-specific secrets here. These are examples only.
